Análise de Variância (ANOVA) – Volume de Vendas por Região

Prof. Dra. Mariana Pereira de Melo (marianapm@usp.br)

1. Por que NÃO usar apenas a média?

Comparar médias simples pode ser tentador, mas isso é perigoso quando queremos tomar decisões reais. A média ignora:

  • variabilidade dos dados — duas regiões podem ter mesma média, mas comportamentos totalmente distintos;
  • incerteza estatística — diferenças pequenas nas médias podem ser apenas “barulho”;
  • erro tipo I — comparar várias médias com testes t separados aumenta falsos positivos;
  • comparações simultâneas, que só a ANOVA resolve corretamente.

A ANOVA resolve tudo isso ao comparar os grupos considerando a relação:

Variação entre regiões / Variação dentro das regiões

3. Gráfico (interpretação)

Este gráfico é gerado por:

ggline(dados_vendas, x="regiaoc", y="vendas",
       add=c("mean_se","jitter"), palette="jco",
       ylab="Vendas (R$ mil)", xlab="Região")

Como interpretar:

  • Pontos (jitter): cada ponto é uma loja real.
  • Barras verticais: erro padrão → variabilidade do grupo.
  • Linhas conectando médias: mostram tendência entre as regiões.

O gráfico sugere que o Sul vende mais e o Leste vende menos — mas isso é só visual. A ANOVA confirma estatisticamente se essa diferença é real.


4. ANOVA

             Df  Sum Sq Mean Sq F value  Pr(>F)    
regiaoc       3  620.1   206.7    17.21  0.00002 ***
Residuals    16  192.0    12.0

Interpretação:

  • F = 17.21: diferença entre regiões MUITO maior que a variação interna.
  • p-value = 0.00002: diferenças são reais, não aleatórias.

➡️ **Conclusão:** pelo menos uma região difere significativamente das demais.


5. Testes de suposição

Normalidade dos resíduos

W = 0.96, p-value = 0.61
✔ Normalidade atendida.

Homogeneidade das variâncias

Bartlett's K² = 1.42, p-value = 0.92
✔ Variâncias iguais entre grupos.

Interpretação:

  • Leste = 28,8 mil (referência)
  • Norte ≈ +8,4 mil
  • Oeste ≈ +4 mil
  • Sul ≈ +15 mil → **maior média**

7. Teste de Tukey

Norte-Leste   8.4   (p = 0.0004)
Sul-Leste    15.2   (p = 0.00001)
Oeste-Leste   4.0   (p = 0.032)
Sul-Norte     6.8   (p = 0.001)

Conclusão do Tukey

  • Sul > todas as regiões
  • Leste < todas as regiões
  • Norte e Oeste são intermediárias

8. Conclusão Geral

A ANOVA mostrou forte evidência de que as médias de vendas diferem entre regiões. O teste de Tukey revelou que:

  • Região Sul é líder absoluta;
  • Região Leste tem desempenho muito inferior;
  • Norte e Oeste ficam no meio do caminho.

Esses resultados permitem decisões estratégicas como:

  • reforçar marketing no Leste;
  • investir mais no Sul para manter o desempenho;
  • entender padrões regionais de comportamento de compra.


9. CÓDIGO R COMPLETO DO ARTIGO


###############################################
# ANÁLISE COMPLETA DE ANOVA - VENDAS POR REGIÃO
###############################################

library(ggplot2)
library(ggpubr)

regiao <- c(
  rep("Norte", 5),
  rep("Sul",   5),
  rep("Leste", 5),
  rep("Oeste", 5)
)

vendas <- c(
  35, 42, 40, 38, 37,   
  45, 48, 43, 44, 46,   
  28, 26, 30, 25, 27,  
  33, 32, 36, 30, 34    
)

regiaoc <- factor(regiao)
dados_vendas <- data.frame(regiaoc, vendas)

ggline(dados_vendas, x="regiaoc", y="vendas",
       add=c("mean_se","jitter"),
       palette="jco",
       ylab="Vendas (R$ mil)",
       xlab="Região")

modelo_vendas <- aov(vendas ~ regiaoc, data = dados_vendas)
summary(modelo_vendas)

shapiro.test(modelo_vendas$residuals)
bartlett.test(vendas ~ regiaoc, data = dados_vendas)

modelo_vendas$coefficients

TukeyHSD(modelo_vendas)

plot(TukeyHSD(modelo_vendas), las=1)
