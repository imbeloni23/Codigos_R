Previsão de Vendas Baseada em Preço e Investimento em Marketing

Uma empresa de varejo deseja prever o volume mensal de vendas (em unidades) a partir de dois fatores:

  • Preço do produto (em R$)
  • Investimento em Marketing (em R$ mil)

O objetivo do estudo é ajustar um modelo de regressão linear múltipla para identificar:

  • Como o preço impacta as vendas;
  • Como o marketing influencia as vendas;
  • Qual variável é mais importante;
  • Se o modelo é capaz de prever vendas futuras.

Por que usar Regressão e não Média Simples?

Comparar médias isoladas (ex.: média de vendas por nível de preço) não permite:

  • Separar efeitos (preço e marketing atuam simultaneamente);
  • Controlar a influência de uma variável enquanto analisa a outra;
  • Medir o impacto marginal de cada variável;
  • Realizar previsões quantitativas do tipo “quanto aumenta se…”;
  • Avaliar significância estatística das variáveis.

A regressão múltipla resolve isso ao modelar a relação linear:

Vendas = β0 + β1·Preço + β2·Marketing

Ajuste do Modelo no R


modelo <- lm(vendas ~ preco + marketing, data=dados)
summary(modelo)

Interpretação

  • Intercepto (118.32): vendas estimadas quando preço = 0 e marketing = 0 (valor estrutural).
  • Preço (–2.38): cada R$ 1 de aumento no preço reduz as vendas em 2,38 unidades, mantido marketing constante.
  • Marketing (+3.10): cada R$ 1 mil investido aumenta as vendas em 3,10 unidades.
  • Ambos p-valores < 0.001: preço e marketing influenciam fortemente as vendas.

Conclusão

A regressão múltipla mostrou que:

  • Marketing e preço são fortemente relacionados às vendas;
  • Marketing tem efeito positivo mais forte que o efeito negativo do preço;
  • O modelo tem excelente ajuste (R² = 0.94);
  • Permite prever vendas futuras de forma quantitativa.

Enquanto a média simples compara grupos isolados, a regressão:

  • controla efeitos simultâneos;
  • mede impacto marginal de cada variável;
  • produz previsões numéricas concretas;
  • é superior para tomada de decisão empresarial.

Código Completo em R


set.seed(123)

n <- 40

preco <- runif(n, 10, 50)
marketing <- runif(n, 5, 50)
vendas <- 120 - 2.5*preco + 3.2*marketing + rnorm(n, 0, 10)

dados <- data.frame(preco, marketing, vendas)

modelo <- lm(vendas ~ preco + marketing, data=dados)
summary(modelo)

novos <- data.frame(
    preco = c(20, 30, 40),
    marketing = c(10, 20, 30)
)

predict(modelo, novos)
