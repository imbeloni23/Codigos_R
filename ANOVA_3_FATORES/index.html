ANOVA 3 Fatores – Fatores que influenciam nas vendas de biscoitos

Luís Gustavo Pimentel Imbeloni (lgimbeloni23@gmail.com)

1. Contexto e Objetivo

Uma empresa deseja entender como três fatores influenciam o volume de vendas de biscoitos (em R$ mil):

  • Região: Norte, Sul, Leste, Oeste;
  • Tamanho da loja: Pequena, Média, Grande;
  • Momento: Antes, Durante e Depois da promoção.

Foram observadas 36 combinações (4 regiões × 3 tamanhos × 3 momentos), com uma loja por combinação.

Por que não usar médias simples?

Comparar médias separadas por região, tamanho ou momento IGNORA que:

  • os fatores atuam simultaneamente;
  • um fator pode mudar o efeito do outro (ex.: promoção só funciona em lojas grandes);
  • não há controle sobre o erro tipo I quando se faz vários testes separados;
  • não é possível detectar interações sem ANOVA fatorial.

A ANOVA de 3 fatores resolve tudo isso ao testar efeitos individuais e interações.

3. Passo a passo do R – com interpretações

3.1 Criando o dataset no R


regiao   <- factor(rep(c("Norte","Sul","Leste","Oeste"), each=9))
tamanho  <- factor(rep(c("Pequena","Média","Grande"), times=12))
momento  <- factor(rep(rep(c("Antes","Durante","Depois"), each=3), times=4))

vendas <- c(
12,15,20,18,23,30,13,17,24,
14,18,22,21,26,34,16,20,27,
10,13,17,14,18,25,11,14,19,
11,16,21,17,22,29,12,17,23
)

dados <- data.frame(regiao, tamanho, momento, vendas)

Interpretação: Criamos três fatores e um vetor de vendas, montando um dataset completo para ANOVA fatorial.

3.2 Executando o modelo


modelo <- aov(vendas ~ regiao * tamanho * momento, data=dados)
summary(modelo)

Interpretação:

  • Momento da Promoção (Antes/Durante/Depois) tem o maior impacto nas vendas.
  • Tamanho da loja também aumenta fortemente o volume vendido.

  • Interações com Região não são significativas → comportamento consistente entre regiões.

4. Conclusão

A ANOVA de três fatores mostrou que não basta olhar médias isoladas. Ela revelou informações que a média nunca mostraria:

  • A promoção impulsiona as vendas, mas **não igualmente**.
  • Lojas grandes capturam muito mais o efeito promocional.
  • Lojas pequenas têm resposta fraca mesmo durante a promoção.
  • O efeito da promoção independe da região (interação fraca).

Ou seja: A promoção funciona — mas funciona para quem tem capacidade de gerar volume.

A média simples jamais revelaria:

  • efeitos individuais;
  • interações entre fatores;
  • como a promoção altera o efeito do tamanho da loja.

A ANOVA fatorial permitiu decompor o fenômeno e entender a dinâmica real das vendas, fornecendo uma base estatística para decisões de marketing e expansão.

5. Código R Completo


# ANOVA FATORIAL 3 FATORES – BISCOITOS

regiao   <- factor(rep(c("Norte","Sul","Leste","Oeste"), each=9))
tamanho  <- factor(rep(c("Pequena","Média","Grande"), times=12))
momento  <- factor(rep(rep(c("Antes","Durante","Depois"), each=3), times=4))

vendas <- c(
12,15,20,18,23,30,13,17,24,
14,18,22,21,26,34,16,20,27,
10,13,17,14,18,25,11,14,19,
11,16,21,17,22,29,12,17,23
)

dados <- data.frame(regiao, tamanho, momento, vendas)

modelo <- aov(vendas ~ regiao * tamanho * momento, data=dados)
summary(modelo)

interaction.plot(momento, tamanho, vendas,
                 main="Interação: Momento × Tamanho",
                 xlab="Momento", ylab="Vendas")

interaction.plot(momento, regiao, vendas,
                 main="Interação: Momento × Região",
                 xlab="Momento", ylab="Vendas")
