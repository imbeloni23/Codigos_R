Análise de Variância (ANOVA) – Volume de Vendas por Região

Luís Gustavo Pimentel Imbeloni (lgimbeloni23@gmail.com)

1. Contexto e objetivo

Uma empresa de venda de automóveis deseja avaliar se o volume médio de vendas mensais (em R$ mil) é diferente entre quatro regiões de atuação: Norte, Sul, Leste e Oeste. Foram selecionadas 5 lojas de cada região e registrado o volume vendido no último mês, totalizando 20 observações.

A técnica utilizada será a Análise de Variância (ANOVA de um fator), cujo objetivo é testar:

  • H0: as médias de vendas são iguais nas quatro regiões;
  • H1: pelo menos uma região possui média de vendas diferente.

A ANOVA é adequada neste caso porque:

  • compara 3 ou mais grupos simultaneamente, de forma controlada;
  • leva em conta a variabilidade dentro de cada região ao avaliar diferenças entre as médias;
  • controla o erro tipo I (não fazemos vários testes t separados).

3. Código R para criação do conjunto de dados

regiao <- c(
  rep("Norte", 5),
  rep("Sul",   5),
  rep("Leste", 5),
  rep("Oeste", 5)
)

vendas <- c(
  35, 42, 40, 38, 37,   # Norte
  45, 48, 43, 44, 46,   # Sul
  28, 26, 30, 25, 27,   # Leste
  33, 32, 36, 30, 34    # Oeste
)

regiaoc <- factor(regiao)

dados_vendas <- data.frame(regiaoc, vendas)
dados_vendas

Interpretação

O código cria o fator regiaoc com quatro níveis (Norte, Sul, Leste, Oeste) e o vetor numérico vendas com os volumes de cada loja. Em seguida, monta-se o data.frame dados_vendas, que será usado em toda a análise. Cada linha do data frame representa a observação de uma loja (região + vendas).

4. Ajuste do modelo de ANOVA

modelo_vendas <- aov(vendas ~ regiaoc, data = dados_vendas)
summary(modelo_vendas)

Saída simulada:

             Df Sum Sq Mean Sq F value  Pr(>F)    
regiaoc       3  620.1  206.7   17.21 0.00002 ***
Residuals    16  192.0   12.0

Interpretação da tabela de ANOVA

  • Df: graus de liberdade. Há 3 para o fator (4 regiões – 1) e 16 para os resíduos (20 observações – 4 grupos).
  • Sum Sq: soma de quadrados. Mede a variabilidade explicada pelo fator (região) e a variabilidade residual.
  • Mean Sq: média dos quadrados (Sum Sq / Df). É a estimativa de variância de cada fonte.
  • F value: razão entre a variância entre grupos e a variância dentro dos grupos.
  • Pr(>F): valor-p associado ao teste F de igualdade das médias.

Como o valor-p associado a regiaoc é 0.00002 < 0.05, rejeita-se H0. Portanto, pelo menos uma região possui média de vendas diferente das demais.

5. Verificação das suposições do modelo

5.1 Normalidade dos resíduos

shapiro.test(modelo_vendas$residuals)

Saída simulada:

Shapiro-Wilk normality test

W = 0.96, p-value = 0.61

Como o valor-p é 0.61 > 0.05, não rejeitamos a hipótese de normalidade. Logo, os resíduos do modelo podem ser considerados aproximadamente normais, atendendo a suposição da ANOVA.

5.2 Homogeneidade das variâncias (Bartlett)

bartlett.test(vendas ~ regiaoc, data = dados_vendas)

Saída simulada:

Bartlett test of homogeneity of variances

Bartlett's K-squared = 1.42, df = 3, p-value = 0.92

O valor-p é 0.92 > 0.05, indicando que não há evidências de variâncias diferentes entre as regiões. A suposição de homocedasticidade (mesma variância em todos os grupos) é satisfeita.

6. Coeficientes do modelo

modelo_vendas$coefficients

Saída simulada:

(Intercept)   regiaocNorte  regiaocOeste    regiaocSul 
      28.8           8.4           4.0          15.2

Interpretação

  • (Intercept): média de vendas da região de referência, que é Leste (28,8 mil).
  • regiaocNorte: diferença Norte – Leste. A região Norte vende, em média, 8,4 mil a mais do que Leste.
  • regiaocOeste: diferença Oeste – Leste (4,0 mil a mais).
  • regiaocSul: diferença Sul – Leste (15,2 mil a mais), indicando a maior média entre as regiões.

Esses coeficientes confirmam a interpretação da ANOVA: Leste apresenta a menor média de vendas, enquanto Sul apresenta a maior.

7. Teste de comparações múltiplas (Tukey)

TukeyHSD(modelo_vendas)

Saída simulada:

$regiaoc
               diff   lwr     upr     p adj
Norte-Leste    8.4   4.1    12.7   0.0004
Sul-Leste     15.2  10.9    19.5   0.00001
Oeste-Leste    4.0   0.3     7.7   0.032
Sul-Norte      6.8   3.1    10.5   0.001
Oeste-Norte   -4.4  -8.1    -0.7   0.020
Oeste-Sul    -11.2 -15.1    -7.3   0.0002

Interpretação

  • As colunas diff mostram a diferença entre as médias de vendas dos pares de regiões.
  • lwr e upr são os limites inferior e superior do intervalo de confiança de 95% para essa diferença.
  • p adj é o valor-p ajustado para comparações múltiplas.

Valores de p adj < 0.05 indicam que as médias desses pares de regiões são significativamente diferentes. No exemplo:

  • Sul possui média de vendas significativamente maior que todas as demais regiões.
  • Leste tem a menor média, diferindo de todas as outras.
  • Oeste é intermediária, ficando acima de Leste e abaixo de Sul.

8. Conclusões finais

  • A ANOVA indicou que há diferenças significativas entre as médias de vendas das regiões (p < 0.05).
  • Os testes de suposições (Shapiro-Wilk e Bartlett) mostraram que o modelo é adequado (normalidade e homogeneidade atendidas).
  • O teste de Tukey permitiu identificar quais regiões diferem entre si:
    • Sul é a região com maior média de vendas;
    • Leste apresenta a menor média;
    • Norte e Oeste têm desempenho intermediário.

Este exemplo mostra como a ANOVA pode ser utilizada para apoiar decisões de negócio, como priorização de investimentos em marketing, logística ou expansão em determinadas regiões.
